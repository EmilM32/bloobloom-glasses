# bloobloom-task

## env

To run this project you need to create a `.env` file in the root directory and add the following variables:

```sh
VITE_API_URL=
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
